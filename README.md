# 浏览器插件 - 快捷网址导航

点击扩展图标显示网站列表，快速打开常用网站。

## 功能

- 显示预设网站列表，点击即可在新标签页打开
- 用户可自定义新增站点（名称、地址、图标、背景色）
- 支持删除站点
- 数据本地持久化保存，刷新后仍保留

## 安装方法

1. 打开 Chrome/Edge 浏览器，进入扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 开启右上角的「开发者模式」

3. 点击「加载已解压的扩展程序」

4. 选择 `browser-extension` 文件夹

## 使用说明

### 打开网站

点击扩展图标，在弹窗中点击目标网站即可打开。

### 新增站点

1. 点击右上角「+ 新增」按钮
2. 填写网站信息：
   - **网站名称**：显示名称（必填）
   - **网站地址**：完整 URL，如 `https://www.example.com`（必填）
   - **图标**：emoji 图标，如 🌐（可选，默认 🌐）
   - **背景色**：图标背景色（可选，默认 #e8f3ff）
3. 点击「添加」保存

### 删除站点

鼠标悬停在站点上，点击右侧出现的「×」按钮即可删除。
